# Java-Studies
Starting basic java
(PT/BR)Iniciando meus estudos em Java! 🚀

Neste primeiro projeto, pratiquei conceitos fundamentais da linguagem, como declaração de variáveis, entrada e saída de dados, estruturas condicionais e interação com o usuário utilizando a classe Scanner.

O programa solicita informações básicas, como nome, idade, peso e se o usuário é estudante. Com esses dados, ele utiliza diferentes tipos de variáveis (String, int, float e boolean) para armazenar e exibir informações de forma organizada. Além disso, implementei uma estrutura condicional (if/else) para verificar e informar se o usuário está matriculado como estudante.

Esse exercício foi importante para compreender como Java lida com diferentes tipos de dados, entrada de informações pelo teclado e tomada de decisões dentro de um programa. Cada novo código representa mais um passo na construção de uma base sólida em programação orientada a objetos e desenvolvimento de software.

(EN)🚀 Starting my Java learning journey!

In this first project, I practiced fundamental Java concepts such as variables, user input/output, conditional statements, and interaction with users through the Scanner class.

The program collects basic information, including the user's name, age, weight, and student status. It uses different data types (String, int, float, and boolean) to store and display information effectively. I also implemented an if/else statement to determine and display whether the user is enrolled as a student.

This exercise helped me better understand how Java handles different data types, keyboard input, and decision-making within a program. Every new project is another step toward building a strong foundation in programming and software development.
